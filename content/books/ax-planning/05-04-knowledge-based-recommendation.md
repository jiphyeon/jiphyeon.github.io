* 스타일, 색상, 브랜드, 사이즈, 가격대 같이 미리 정의된 조건을 사용자가 선택하면 해당 조건에 맞는 상품을 추천하는 방식
* 사용자 행동 데이터 없이도 작동해서 콜드 스타트 문제의 영향을 덜 받음
* 단순 메타데이터 필터링만 쓸 수도 있지만, TF-IDF와 K-Means를 결합하면 추천의 다양성과 정확성이 크게 향상됨
* 한계: 사용자가 조건을 직접 선택해야만 추천이 작동하고, 조건과 약간 다른 유사 상품은 추천하지 못함

<b>메타데이터 필터링 방식</b> : 상품 메타데이터(스타일, 색상, 브랜드, 사이즈, 가격대 등)를 기반으로 사용자 선택 조건과 정확히 일치하는 상품만 필터링. 구현은 단순하지만 추천 범위가 좁음.

<b>TF-IDF 결합 방식</b> : 상품 설명, 리뷰에서 연관 용어의 중요도를 계산해 메타데이터 태그가 없어도 의미적으로 유사한 상품을 추천. 태그가 없는 상품까지 추천 범위가 확장됨

<b>K-Means 클러스터링 결합 방식</b> : 상품 설명, 속성, 사용자 행동 패턴을 기반으로 유사한 상품을 자동으로 그룹화. 추천 다양성과 정확성 향상
